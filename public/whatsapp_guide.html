Cara Setup & Konfigurasi

1. Pilih Provider Gateway

Masuk ke Settings > WhatsApp Center. Kami merekomendasikan Wablas atau Fonnte.

Rekomendasi: Wablas / Fonnte

2. Input API Token & Server URL

Dapatkan API Token dari dashboard provider Anda. Paste ke kolom sesuai modul (Sales atau Purchasing) lalu klik Save Settings.

3. Setup Webhook (PENTING!)

Salin URL Webhook di bawah ini sesuai modul dan tempelkan pada dashboard provider Anda (Fonnte/Wablas) agar pesan masuk dapat direspon AI otomatis.

https://jicos.jidoka.co.id/api/whatsapp/webhook
