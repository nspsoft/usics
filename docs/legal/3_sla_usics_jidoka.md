### PASAL 2 - KATEGORI BANTUAN TEKNIS DAN BATAS RESPON (RESPONSE TIME)
Dukungan peladen dan perbaikan *software* dari Kelompok Dukungan IT *(IT Helpdesk USICS)* akan diberikan merujuk kuota Respon Matriks prioritas:
- **PRIORITAS 1 (Kritis) - Target Tanggapan < 2 Jam:**
  Seluruh Jaringan Server Utama jatuh/ *Down*, database tidak dapat dijangkau yang memberhentikan 100% produksi. Target Perbaikan Normalisasi (Recovery Time): *Maks. 1x24 Jam Kerja*.
- **PRIORITAS 2 (Medium) - Target Tanggapan < 8 Jam:**
  Satu menu bermasalah mengunci divisi (misal: Modul Pembelian eror) tapi modul Penjualan lain hidup. Target Perbaikan Normalisasi: *Maks. 2 Hari Kerja*.
- **PRIORITAS 3 (Rendah) - Target Tanggapan < 2 Hari:**
  Pertanyaan pengoperasian dari pengguna, ketidakmerataan garis UI ringan, maupun perubahan *format template Printer PDF*.
